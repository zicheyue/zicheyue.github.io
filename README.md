# 李军朋教授课题组网站

这是一个使用 HTML、CSS 和 JavaScript 创建的静态个人/课题组网站。页面面向企业合作和研究生招生，重点展示大模型、具身智能、机械臂智能控制、高炉大模型和工业智能建模方向。

## 文件结构

```text
personal_website/
├── index.html
├── style.css
├── script.js
├── assets/
│   └── thumbs/
└── *.mp4
```

## 页面内容

- 首页与核心方向
- 基于大模型的机械臂智能控制演示视频
- 企业合作方向
- 负责人简介
- 重要论文成果
- 科研项目
- 招生与学生培养
- 联系方式

## 在 Windows 上预览

方法一：直接打开文件

1. 在文件资源管理器中进入本项目文件夹。
2. 双击 `index.html`。
3. 页面会在默认浏览器中打开。

方法二：使用 VS Code Live Server

1. 用 VS Code 打开本项目文件夹。
2. 安装扩展 `Live Server`。
3. 右键点击 `index.html`，选择 `Open with Live Server`。
